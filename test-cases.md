#### TC-1.Авторизация пользователя с валидным логином и паролем (управление мышью)

<details>

**Предусловия:** Открыть сайт https://opensource-demo.orangehrmlive.com и выйти из аккаунта.

| # | Шаги | Ожидаемый результат |

| 1 | Открыть сайт OrangeHRM | Открывается страница с формой авторизации |

| 2 | Ввести в поле Username "Admin" | Логин отображается в поле |

| 3 | Ввести в поле Password "admin123" | Символы пароля маскируются |

| 4 | Нажать кнопку Login | Успешная авторизация и автоматический переход на страницу '“Dashboard” |


**Постусловия:**

| # | Шаги | Ожидаемый результат |
| 1 | В user menu(справа вверху фотография аккаунта с именем и значком выпадающего меню) вызвать выпадающий список и нажать на “Logout” | Открывается страница с формой авторизации |
