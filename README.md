# OrangeHRM Manual Testing Project

## Описание проекта

Учебный проект по ручному тестированию системы управления персоналом OrangeHRM.

## Цель

Проверка авторизации, создание и редактирование профиля. 

## Выполненные работы

- создан чек-лист;
- разработано 86 тест-кейсов;
- оформлено 11 баг-репортов.

## Используемые инструменты

- Jira
- Git
- GitHub
- DevTools
- Miro

## Структура репозитория

test-cases/ — тест-кейсы

checklists/ — чек-листы

bug-reports/ — баг-репорты
