# OrangeHRM Manual Testing Project

## Описание проекта

Учебный проект по ручному тестированию системы управления персоналом OrangeHRM.

## Цель

Проверка основных пользовательских сценариев:
- авторизация;
- управление сотрудниками;
- поиск;
- роли пользователей.

## Выполненные работы

- составлен тест-план;
- разработано 60 тест-кейсов;
- создано 4 чек-листа;
- оформлено 12 баг-репортов.

## Используемые инструменты

- Jira
- Git
- GitHub
- DevTools
- Miro

## Структура репозитория

test-plan/ — тест-план

test-cases/ — тест-кейсы

checklists/ — чек-листы

bug-reports/ — баг-репорты
